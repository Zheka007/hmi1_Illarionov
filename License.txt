Теперь при запуске CMake:
1. Автоматически скачается SFML 3.0.2 с GitHub
2. Автоматически скачается Google Test 1.14.0
3. Библиотеки будут собраны локально
4. Проект слинкуется с ними

**Запусти так:**
```bash
# Удали старую папку build если есть
rm -rf build

# Создай новую и собери
mkdir build
cd build
cmake -G "Ninja" ..
cmake --build .
